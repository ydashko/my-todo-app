# Todo App

A simple Todo app built with Next.js, Tailwind CSS, and JSONPlaceholder API.

## Features
- View a list of todos
- Add a new todo
- Delete a todo

## Tech Stack
- Next.js (Pages Router)
- Tailwind CSS
- Axios
- TypeScript

## How to run

```bash
git clone https://github.com/your-username/todo-app.git
cd todo-app
npm install
npm run dev
